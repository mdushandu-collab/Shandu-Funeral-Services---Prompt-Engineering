# Shandu-Funeral-Services---Prompt-Engineering
Designed and developed a full-stack funeral services website with structured pricing tables, 6-page user journey mapping, and conversion-focused CTAs. Showcases skills in requirements gathering, data organisation, UX design for sensitive contexts, responsive UI development, and translating business needs into functional digital solutions.

**Skills Demonstrated**

**1. Business Analysis & Requirements Engineering**

- Conducted thorough stakeholder requirements gathering to translate business objectives into functional specifications
- Developed a comprehensive 6-page site architecture (Home, Services, Plans, Gallery, Contact Us, About Us) with clearly defined user journeys
- Created detailed acceptance criteria for each feature, ensuring alignment between business needs and technical implementation
- Identified target market segments and mapped product offerings accordingly, demonstrating understanding of customer segmentation principles


**2. Data Organisation & Structuring**

- Designed and structured a multi-variable pricing matrix comprising 10 distinct funeral plans
- Organised complex data relationships across five dimensions: Age Band, Premium Range, Cover Amount, Plan Type, and Target Segment
- Implemented logical data categorisation enabling filtering and sorting functionality for enhanced user experience
- Demonstrated ability to transform raw tabular data into user-friendly, decision-support interfaces


**Sample Data Structure Managed:**

| Dimension | Complexity
|-----|-----
| Plan Types | 10 distinct products
| Age Bands | 8 unique ranges (18-35 through 25-65)
| Premium Tiers | Variable ranges (R26 - R200/month)
| Cover Amounts | R5,000 - R100,000
| Market Segments | 10 distinct customer profiles


**3. User Experience (UX) Design & Information Architecture**

- Developed intuitive navigation structures with clear call-to-action pathways
- Designed service categorisation using clickable folder/card components ("Purchase Plan," "How To Claim," "Burial and Cremation")
- Created conversion-focused user flows guiding visitors from awareness to purchase intent
- Applied context-appropriate design sensitivity—balancing professionalism with compassion for emotionally charged subject matter
- Specified inclusive imagery requirements reflecting South Africa's diverse demographic composition

**4. Technical Implementation & Front-End Development**

- Built responsive, mobile-first web application using modern development frameworks
- Implemented interactive UI components including:

- Dynamic navigation with sticky header functionality
- Interactive background colour transitions
- Hover effects and smooth scroll behaviours
- Lightbox gallery with category filtering
- Form validation and submission handling

- Integrated external services: Google Maps embed, WhatsApp chat functionality, mailto/tel links
- Applied accessibility standards (WCAG 2.1 AA compliance considerations)

**5. Content Strategy & Brand Communication**

- Developed brand identity specifications including logo conceptualisation (Tree of Life symbolism) and colour psychology application
- Crafted vision and mission statements aligned with business positioning
- Wrote emotionally intelligent copy addressing sensitive subject matter while maintaining professional tone
- Ensured consistent brand voice across all touchpoints

**6. Project Documentation & Communication**

- Produced comprehensive technical documentation suitable for developer handoff
- Demonstrated ability to communicate complex requirements clearly to both technical and non-technical stakeholders
- Created structured specifications following industry-standard documentation practices
